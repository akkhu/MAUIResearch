# MAUIResearch

This repository contains a sample .NET MAUI application demonstrating Syncfusion PDF Viewer annotations using the MVVM pattern.

The project is located in the `AnnotationApp` folder. It showcases how to load a PDF document and provide built-in annotation features.

## Building the app

Install the required .NET MAUI workload and restore NuGet packages. Then open the solution in Visual Studio or run `dotnet build`.
